# Series Complete for Plex - Home Assistant App

Analyze your Plex TV library for missing episodes. Connects to your Plex server over the network via REST API.

## Installation

1. Open Home Assistant
2. Go to **Settings** > **Apps** > **App Store**
3. Click the three-dot menu (top right) > **Repositories**
4. Add: `https://github.com/akustikrausch/series-complete-for-plex`
5. Find "Series Complete for Plex" in the store and click **Install**
6. Configure the app settings (Plex URL, Plex Token, API keys)
7. Start the app
8. Access via the Home Assistant sidebar

## Configuration

See the **Documentation** tab in the app panel for full setup instructions.
