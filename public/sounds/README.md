# Sound Files

Place notification sounds here:
- `established.mp3` - Success/completion sound
- `error.mp3` - Error notification sound
- `warning.mp3` - Warning notification sound

You can use any short audio file (1-2 seconds recommended).

## Free Sound Resources:
- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/

## Recommended sounds:
- Success: Short chime or bell
- Error: Subtle error tone
- Warning: Soft alert sound
