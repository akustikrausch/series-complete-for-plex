Overview

What is Series Complete for Plex?

Series Complete for Plex is an advanced open-source analysis tool specifically designed for Plex Media Server users who want to maintain a complete and organized TV series collection. It provides deep insights into your media library by connecting directly to your local Plex database and cross-referencing with multiple online databases to identify missing episodes, incomplete seasons, and series that need attention.

Unlike simple file scanners, Series Complete for Plex understands the actual content of your library, recognizes different naming conventions, handles multi-part episodes, and provides accurate completeness metrics based on official episode counts from trusted sources like TMDb, TheTVDB, and even uses AI verification through OpenAI when needed.

Available APIs

1. TMDb (The Movie Database) - STRONGLY RECOMMENDED

The most comprehensive and reliable source for TV series data. Provides series info, episode counts, and completion status.

Step-by-Step Setup:
  1. Go to themoviedb.org
  2. Click "Join TMDb" (top right) and create free account
  3. Confirm your email address
  4. Go to Settings → API → Create → Developer
  5. Accept the terms and conditions
  6. Fill out the form: Application Name: "Series Complete for Plex", URL: "http://localhost:3000", Summary: "Personal TV series tracker"
  7. Copy the "API Key (v3 auth)" - this is what you need!

Cost: Completely FREE with generous limits (thousands of requests per day)

2. TheTVDB - RECOMMENDED

Excellent backup source with detailed episode information and air dates. Perfect complement to TMDb.

Step-by-Step Setup:
  1. Go to thetvdb.com
  2. Click "Register" and create free account
  3. Confirm your email address
  4. Go to Dashboard → API Access
  5. Click "Create API Key"
  6. Fill out: API Key Name: "Series Complete for Plex", Description: "Personal use for TV series tracking"
  7. Copy your API key

Cost: FREE for personal use

3. OpenAI - OPTIONAL

AI-powered fallback for very obscure series that other APIs don't know. Only used when TMDb and TheTVDB fail.

Step-by-Step Setup:
  1. Go to platform.openai.com
  2. Create account or sign in
  3. Go to API Keys section
  4. Click "Create new secret key"
  5. Name it "Series Complete for Plex" and copy the key
  6. IMPORTANT: Add billing info and set usage limits!

Cost: Pay-per-use (~$0.001 per series analysis)

4. OMDb - OPTIONAL

Additional metadata source for poster images and IMDb ratings. Nice to have but not essential.

Step-by-Step Setup:
  1. Go to omdbapi.com/apikey.aspx
  2. Select "FREE! (1,000 daily limit)"
  3. Enter your email address
  4. Check your email and click activation link
  5. Copy your API key from the email

Cost: FREE (1,000 requests per day)

Troubleshooting

Problem: Database not found

Solution:

  • Ensure Plex Media Server is running
  • Check path in .env file
  • Default Windows path: C:\Users\[Name]\AppData\Local\Plex Media Server

Problem: Analysis finds no metadata

Solution:

  • Check API configuration in settings
  • Test each API individually with "Test" button
  • Ensure API keys are valid

Problem: Browser becomes slow

Solution:

  • Use "Clear Cache" in settings
  • Close other browser tabs
  • Use Chrome or Edge for best performance

Smart Retry System

Automatic Failure Recovery

Series Complete for Plex automatically retries failed operations using intelligent exponential backoff:

  • API requests: Up to 3 retries with exponential backoff
  • Network requests: Up to 5 retries for connection issues
  • Database operations: Up to 2 retries with longer delays

Circuit Breaker Protection

Prevents cascading failures by temporarily disabling problematic endpoints:

  • Opens circuit after 5 consecutive failures
  • Automatically resets after 1 minute cooling period
  • Half-open state tests endpoint before full recovery

Configurable Settings

Access retry configuration via the "Retry Settings" button (bottom-left corner):

Configuration Tab
  • Adjust retry counts per operation type
  • Configure delays and timeouts
  • Enable/disable jitter
Monitoring Tab
  • View active retry operations
  • Monitor circuit breaker states
  • Review retry statistics
